## 7. ⚡ Cache Aware Scheduling Coming to Linux 7.2 — Big Wins for AMD EPYC/Zen 5 Servers

Phoronix benchmarked it on AMD EPYC Zen 5 systems and found significant performance improvements across PostgreSQL, Valkey (Redis fork), and network workloads. This is a major optimization for bare-metal database and caching servers running on AMD hardware.

**Source:** https://www.phoronix.com/review/cache-aware-scheduling-hedt

## 8. 💾 Meta Revives CacheLib After 2 Years — Fighting Sky-High DRAM Prices in Data Centers

Meta (Facebook) released a major new version of **CacheLib** — their open-source caching engine — after a two-year hiatus. The timing is no coincidence: DRAM prices have skyrocketed due to AI demand, and better caching directly reduces memory costs in bare-metal deployments. CacheLib originally targeted NVMe-based caching; the new release expands its capabilities for today's cost-constrained data centers.

**Source:** https://www.phoronix.com/news/CacheLib-2026.05.25

## 9. 🔒 Intel TDX Runtime Updates Coming to Linux 7.2 — No More Reboots for Security Patches

Intel's **Trusted Domain Extensions (TDX)** runtime update feature is expected to land in Linux 7.2, allowing confidential computing VM security modules to be updated **without server reboot**. For bare-metal operators running confidential VMs, this eliminates a major operational headache — applying security patches without draining and restarting physical hosts.

**Source:** https://www.phoronix.com/news/Intel-TDX-Runtime-Update-7.2

## 🧠 Bare-Metal Trend Summary

- **ARM Bare-Metal Is Getting Real** 🔥 — NVIDIA Vera's benchmark results show ARM server CPUs are no longer niche. With custom Olympus cores competing directly with Xeon and EPYC, expect a wave of ARM bare-metal offerings in the next 12-18 months.

- **GPU Bare-Metal Dominates Infrastructure Investment** 💰 — Lambda's $1B credit facility and bare-metal instance launch underscore that the market is voting with dollars: GPU-accelerated bare-metal for AI/ML workloads is where the growth is. BlueField DPUs are becoming the standard for hardware-level isolation.

- **Liquid Cooling Goes Mainstream** ❄️ — Direct-to-chip liquid cooling is no longer experimental. Equinix's deep-dive article and the APC PowerForge announcement (3D-printed rack solutions with Dell and NVIDIA) show the industry is standardizing on liquid cooling for high-density bare-metal deployments.

- **Cloud Repatriation to Bare-Metal Is Accelerating** 🔄 — Equinix's infrastructure experts are explicitly making the case that the cloud-first pendulum is swinging back. Enterprise operators are re-evaluating dedicated hardware for cost predictability, compliance, and raw performance — especially for AI inference and database workloads.

- **Linux 7.2 Brings Major Server Optimizations** ⚡ — Cache Aware Scheduling for AMD Zen 5, Intel TDX runtime-update capability, and other kernel improvements landing in the upcoming Linux release will directly benefit bare-metal server operators running their own hardware.
