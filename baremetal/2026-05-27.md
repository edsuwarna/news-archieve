## 3. 🍎 AWS Launches EC2 M3 Ultra Mac Instances — Bare-Metal Mac in the Cloud

**Summary:** AWS announced general availability of EC2 M3 Ultra Mac instances, built on Apple M3 Ultra Mac Studio with a 28-core CPU, 60-core GPU, 32-core Neural Engine, and 256GB unified memory. Compared to the previous M4 Max instances, M3 Ultra delivers 2x the unified memory, 1.75x more CPU cores, 1.5x more GPU cores, and 2x the Neural Engine cores. These bare-metal Mac instances let Apple developers run more Xcode simulators in parallel and accelerate on-device ML workflows — a key BMaaS (Bare Metal as a Service) offering.

**Source:** https://aws.amazon.com/blogs/aws/aws-weekly-roundup-aws-transform-at-1-year-claude-platform-on-aws-ec2-m3-ultra-mac-instances-and-more-may-18-2026/

## 4. ⚡ Supermicro Ships 36-Core Intel Xeon 6 SoC Server with Dual 100GbE

**Summary:** ServeTheHome reviewed the Supermicro SYS-112D-36C-FN3P, a short-depth 1U server packing a 36-core Intel Xeon 6 SoC (Granite Rapids-D) with two QSFP28 100GbE ports. At ~15.7 inches deep, this is purpose-built for edge racks and compact data center deployments where depth is a constraint. The system replaces 8×25GbE with 2×100GbE for higher-bandwidth networking in the same front-I/O form factor. A strong contender for edge computing on bare-metal.

**Source:** https://www.servethehome.com/supermicro-sys-112d-36c-fn3p-review-a-36-core-intel-xeon-6-soc-server-with-2x-100gbe/

## 5. 🔬 Equal1 Debuts Single-Rack Quantum Computer at Dell Tech World 2026

**Summary:** Irish startup Equal1 demonstrated a quantum computer that fits entirely within a standard 19" rack — a first for the industry. Using silicon spin qubits fabricated with standard CMOS processes, the system integrates classical Dell PowerEdge servers for control, RF electronics, and a cryogenic cooling cylinder (below 1 Kelvin) all in one rack. This eliminates the giant cooling plants previously required for quantum computers, enabling quantum to be deployed alongside traditional bare-metal servers in existing data centers.

**Source:** https://www.servethehome.com/equal1-single-rack-quantum-computer-at-dell-tech-world-2026/

## 8. 📦 Dell PowerStore Elite: 5.8PB Storage Appliance with PCIe Gen5 and QLC Parity

## 🔥 Bare-Metal Trend Summary

- **Next-Gen x86 Server Wave Arriving H2 2026** — AMD EPYC Venice (N2 node, 256 cores) and Intel Xeon 6 SoC / Diamond Rapids are driving the most significant server refresh cycle in years, with 70%+ generational performance gains. Dell's 18th-gen PowerEdge and Supermicro's edge-optimized Xeon 6 SoC servers lead the charge.
- **ARM Bare-Metal Gains Real Traction** — NVIDIA Vera CPU benchmarks show ARM finally delivering server-class performance that rivals x86, while AWS continues expanding its Apple silicon-based bare-metal Mac instances. The ARM server ecosystem is no longer theoretical.
- **Liquid Cooling Goes Mainstream** — With single racks now exceeding 480kW (Dell's IR7000), direct-to-chip liquid cooling is moving from specialized HPC into mainstream data center design. Equinix's deep-dive signals that colo providers are preparing for widespread liquid-cooled bare-metal deployments.
- **Cloud Repatriation & Colocation Renaissance** — Multiple signals (Equinix articles, Ubicloud's growth, the "Cloud Isn't Right for Everything" narrative) point to enterprises re-evaluating cloud-first strategies and returning predictable workloads to dedicated hardware. BMaaS and colocation providers stand to benefit.
- **AI Hardware Demand Reshaping Everything** — From agentic AI requiring edge-deployed bare-metal to 5.8PB storage appliances for training datasets, the AI boom continues to drive innovation in dedicated server hardware, cooling, storage density, and interconnect bandwidth.
