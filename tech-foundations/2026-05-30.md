## 🌐 **CNCF (Cloud Native Computing Foundation)**

### 🚀 Kubernetes v1.36 "Haru" Released!
The biggest CNCF project just dropped v1.36 with some seriously exciting features:
- **User Namespaces finally GA** 🎉 — major security boundary improvement
- **Declarative Validation graduates to GA** — policy-as-code just got real
- **PSI Metrics graduates to GA** — better pressure stall monitoring
- **Pod-Level Resource Managers (Alpha)** — fine-grained resource control
- **Tiered Memory Protection with Memory QoS** — for memory-constrained workloads
- **Mutable Pod Resources for Suspended Jobs (beta)** — edit suspended jobs
- **ExternalIPs removed** — long-deprecated feature officially gone
- **Route Sync metrics in Cloud Controller Manager** — new metric
- **DRA (Dynamic Resource Allocation)** — more drivers, new features
